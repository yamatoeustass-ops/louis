## 📋 Step-by-Step (100% Free)

### Part 1: Upload to GitHub (5 minutes)

**Step 1: Create GitHub Account**
```
1. Go to: https://github.com/signup
2. Enter email
3. Create password
4. Verify email
5. Done! (2 minutes)
```

**Step 2: Create Public Repository**
```
1. Go to: https://github.com/new
2. Repository name: viral_caption
3. Description: AI Transcription for Shorts, Reels & TikTok
4. ✅ PUBLIC (important - free Actions!)
5. DO NOT initialize with README
6. Click "Create repository"
```

**Step 3: Upload Files**
```
1. On repository page, click "uploading an existing file"
2. Drag ALL files from mobile_transcription folder
3. Wait for upload (2-5 minutes)
4. Commit message: "Initial commit"
5. Click "Commit changes"
```

---

### Part 2: Enable GitHub Actions (1 minute)

**Step 4: Enable Actions**
```
1. In your repo, click "Actions" tab
2. You'll see "Build Android APK" workflow
3. Click "I understand my workflows, go ahead and enable them"
4. Click "Enable workflow" button
```

**Step 5: First Build Starts**
```
1. Build starts automatically!
2. Wait 15-20 minutes
3. Go get coffee ☕
4. Come back to green checkmark ✅
```

---

### Part 3: Download APK (2 minutes)

**Step 6: Get APK from Artifacts**
```
1. Click on the completed build (green checkmark)
2. Scroll to "Artifacts" section at bottom
3. Click "apk-files" to download
4. ZIP file downloads to your computer
5. Extract ZIP → Get APK files
```

**What's in the ZIP:**
- `app-release.apk` (universal, ~50-60 MB) ← Use this!
- `app-arm64-v8a-release.apk` (modern phones, ~20 MB)
- `app-armeabi-v7a-release.apk` (older phones, ~18 MB)

---

### Part 4: Install on Phone (3 minutes)

**Transfer APK to Phone:**

**Method A: USB Cable**
```
1. Connect phone to computer (USB)
2. Copy APK to phone's Download folder
3. On phone: Files → Downloads → Tap APK
4. Enable "Install unknown apps" if needed
5. Install
```

**Method B: Google Drive (No Cable)**
```
1. Go to: drive.google.com
2. Click "New" → "File upload"
3. Upload the APK
4. On phone: Open Drive app
5. Download APK
6. Install
```

**Method C: Email**
```
1. Email APK to yourself
2. Open email on phone
3. Download attachment
4. Install
```

## 🆘 Troubleshooting

### "Actions not enabled"
```
Solution:
1. Go to repo Settings
2. Actions → General
3. Allow all actions
4. Save
```

### "Build failed"
```
Solution:
1. Click on failed build
2. Read error logs
3. Fix the file mentioned
4. Commit fix
5. New build starts automatically
```

### "Can't download artifact"
```
Solution:
1. Make sure build completed (green checkmark)
2. Scroll to bottom of build page
3. Artifacts section appears after success
4. Click to download
```

### "Build takes too long"
```
Normal times:
- First build: 20-25 minutes
- Later builds: 12-18 minutes

If longer:
- Check build logs for errors
- May be stuck - cancel and retry
```
