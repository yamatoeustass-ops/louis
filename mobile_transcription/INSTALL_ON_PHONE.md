# 📱 Install ViralCaption AI on Huawei Phone

## Choose Your Situation:

---

## ❌ I DON'T Have Flutter Installed

### → Use Cloud Build (10 minutes, No Installation!)

**Step 1: Upload to GitHub**
```
1. Create GitHub account: https://github.com/signup
2. Create new repository named "viral_caption"
3. Upload all files from mobile_transcription folder
```

**Step 2: Build on Codemagic**
```
1. Go to: https://codemagic.io/
2. Sign in with GitHub
3. Click "Add repository"
4. Select "viral_caption"
5. Click "Start build"
6. Wait ~10 minutes
7. Download APK from build artifacts
```

**Step 3: Install on Phone**
```
1. Transfer downloaded APK to Huawei phone
2. Files app → Downloads → Tap APK
3. If prompted: Settings → Security → Enable "Install unknown apps"
4. Tap Install
5. Done! Open ViralCaption AI
```

📖 **Full guide:** `NO_FLUTTER_GUIDE.md`

---

## ✅ I HAVE Flutter Installed

### → Build Locally (2-5 minutes)

**Windows:**
```
Double-click: BUILD.bat
```

**Mac/Linux:**
```
Double-click: BUILD.sh
```

**Then:**
```
1. Download page opens automatically
2. Click "Download APK" button
3. Transfer APK to phone
4. Install on phone
```

📖 **Full guide:** `INSTALL_GUIDE.md`

## 🔧 Quick Fixes

| Problem | Solution |
|---------|----------|
| "Flutter not found" | Use cloud build (Codemagic) |
| "Build failed" | Run `flutter doctor`, fix issues |
| "Can't install APK" | Enable "Install unknown apps" in Settings → Security |
| "App won't open" | Check Android version (need 7.0+) |
